# Emoji-Klonlama
Herkese Selamlar bugün öyle kendi kafamın estiği şekilde Emoji klonlama yaptım. Server klonlama ile aynı mantıkta 

> [!NOTE]
> Python sürümünüz 3.11 olacaktır. Kurmadan evvelde Add to Path Seçeneğini aktif ediniz.

> [!TIP]
> Kendi tokeninizi girmeniz gerekmekte Bot tokeni ile kullanmayınız.
> Artı olarak emojileri çekerken tamamen size yeni bir klasör oluşturarak yapmaktadır.

> [!CAUTION]
> Tamamen Lisanslı aksi taktirde başka bir platform ya da Video üzeri görülürse Yasal süreçler başlayacaktır
